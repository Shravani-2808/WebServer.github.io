What is Web Server


A web server is computer software and underlying hardware that accepts requests via HTTP, the network protocol is created to distribute web pages or its secure variant HTTPS.

A user-agent, probably a web browser or a web crawler, initiates communication by making a request for a specific resource using HTTP. Immediately, the server responds with the content of that resource or an error message. The server can also accept and store resources sent by the user-agent end if configured to do so.

We use 'http.request' to send the data to the server and await for the response. The response is stored in the 'req' variable, and upon error, it is logged into the console. On successful transmission, the data is posted to the server.
